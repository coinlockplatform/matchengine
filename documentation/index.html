File/Folder Structure

The files are placed in organized folders so to make it easy for you to find files where you will need to do customizations or change other stuffs easily.

Below is the folders and details of what they are for

  • CSS: This folder contains all style files or stylesheets.
  • Fonts: This folder contains all the fonts used for in the Template.
  • Js: This folder contains all the scripts
  • Img: This folder contains all the images

Contact Form

The contact form uses the PHP file for sending emails via contact form. By default, the email queries from the form goes to a dummy email address. To change the email to your email address which will receive the email queries from the contact form. Please follow the following steps.

  • Open the "form-process.php" file which can be found in the main directory in any text editor.
  • In the second line, /* Set e-mail recipient */ Change the "$myemail = "PUT YOUR EMAIL HERE";" to your email.
